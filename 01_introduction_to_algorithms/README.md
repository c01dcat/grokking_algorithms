# 算法简介

* 算法是一组完成任务的指令，任何代码片段都可视为算法。
* NP 完全问题：多项式复杂程度的非确定性问题。
* 一般而言，对于包含 n 个元素的列表,用二分查找最多需要 log<sub>2</sub>n 步，而简单查找最多需要 n 步。
* 简单查找最多需要查找的次数与列表长度相同, 这被称为线性时间（linear time） 。二分查找的运行时间为对数时间（或 log 时间） 。

## 大 O 表示法

大 O 表示法是一种特殊的表示法，指出了算法的速度有多快，计算的是操作数。说的是最糟的情形。

5 种常见的大 O 运行时间：

* O(log<sub>2</sub>n)，也叫对数时间，例如二分查找。
* O(n)，也叫线性时间，例如简单查找。
* O(n * log<sub>2</sub>n)，例如快速排序。
* O(n<sup>2</sup>)，例如选择排序。
* O(n!)，即阶乘时间，例如旅行商问题的解决方案。

算法的速度指的并非时间，而是操作数的增速。
